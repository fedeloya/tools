Digital Enabler Test Page for Smart Site Metadata Crawler

This test page includes ld+json metadata embedded within the body of the HTML. It simulates a realistic structure for pages using rich structured data to improve discoverability by search engines or internal metadata crawlers.

The metadata refers to the IBM Maximo Price Estimator tool and is part of the Digital Enabler initiative. It helps users explore IBM Maximo pricing options with estimated discounts for SaaS solutions.

Note: The metadata script is intentionally placed within the <body> tag to match the final integration format used across IBM’s dynamic content pages.
